## 一、配置公众号自定义菜单（最快方式）

### 操作步骤：

1. 登录微信公众号后台：https://mp.weixin.qq.com
2. 左侧菜单 → **内容与互动** → **自定义菜单**
3. 添加菜单项：
   - **菜单名称**：📚 产业动态（或自定义）
   - **菜单内容**：选择「跳转网页」
   - **页面地址**：`https://291e810cb93670.lhr.life`

4. 点击「保存并发布」

✅ 完成！粉丝点击菜单即可打开翻页电子书架。

## 二、（可选）去除微信顶部安全提示

默认情况下，未加入「业务域名」白名单的链接在微信中打开会显示"非微信官方网页"提示。

### 去除方法：

1. 公众号后台 → **设置与开发** → **公众号设置** → **功能设置**
2. 找到「业务域名」，点击「设置」
3. 下载验证文件 `MP_verify_xxx.txt`
4. **将该文件放入本项目 `wechat-verify/` 目录**

文件放入后无需重启服务器，微信验证会自动通过。

5. 在业务域名中添加：`291e810cb93670.lhr.life`

> 验证文件路由已内置在服务器中，支持所有 `MP_verify_*.txt` 格式。

## 四、隧道保活

如果公网链接失效（电脑重启/网络断开），重新运行：

```bash
cd "/Users/guyue/Desktop/mac电脑同步文件/办公助手/flipbook"
ssh -R 80:localhost:3456 nokey@localhost.run
```

获取新的公网地址后，更新公众号菜单链接即可。

在 https://localhost.run 注册并添加 SSH 公钥，可获得固定子域名，
无需每次更新菜单链接。
